The Beatles FanPage:

An immersive tribute to the Fab Four. Built with a focus on high-end animations and smooth user experience.

🌐 Live Demo: https://zingy-monstera-47fa09.netlify.app

🛠️ Stack
Core: Next.js 16 (App Router) + TypeScript
Motion: GSAP (ScrollTrigger) + Lenis (Smooth Scroll)
Styling: Tailwind CSS

⚡ Key Features
Smart Header: Auto-hiding navigation synced with scroll direction.
Fluid Motion: Cinematic parallax and smooth-scroll integration.
Responsive: Fully optimized for all device sizes.

🚀 Setup
git clone https://github.com/FeyoChavez/portfolio.git
npm install --legacy-peer-deps
npm run dev
