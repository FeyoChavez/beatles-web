# The Beatles FanPage

An immersive tribute to the Fab Four. Built with a focus on high-end animations and smooth user experience.

[🌐 Live Demo](https://zingy-monstera-47fa09.netlify.app)


## 🛠️ Stack
* **Core:** Next.js 16 (App Router) + TypeScript
* **Motion:** GSAP (ScrollTrigger) + Lenis (Smooth Scroll)
* **Styling:** Tailwind CSS


## ⚡ Engineering Highlights
* **Adaptive Header:** GSAP-driven navigation that reacts to scroll velocity and direction.
* **Performance:** Optimized for React 19, ensuring smooth 60fps animations.
* **UX:** Integrated Lenis for a consistent inertial scrolling experience across browsers.


## 🚀 Setup

* `git clone https://github.com/FeyoChavez/beatles-web.git`
* `npm install --legacy-peer-deps`
* `npm run dev`


## 📝 Project Note

This project was developed as a study in combining high-end web animations with modern engineering practices, focusing on performance and accessible UI/UX.
